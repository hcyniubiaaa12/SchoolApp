# 高校校园助手 APP — 前端视觉优化设计文档

> 基于「青春活力 · 色彩律动」设计方向，在保持现有配色系统和字体规范的前提下，为 APP 注入独特的视觉签名。
> 日期：2026-06-27 | 状态：已批准

---

## 一、设计目标

| 维度 | 说明 |
|------|------|
| 核心诉求 | 在保持现有功能不变的前提下，给 APP 一个独特、统一的视觉身份 |
| 风格关键词 | 青春活力、色彩律动、分类清晰、视觉统一 |
| 不变约束 | 字体大小（float.json 原样保留）、主色系（color.json 原样保留） |
| 签名元素 | 「分类色彩标识系统」— 每个模块根据内容类型获得独特的色彩标记 |

## 三、签名元素规范

### 3.1 彩色顶部色条

所有白底卡片在顶部添加 3px 的彩色色条，颜色根据卡片内容类型确定。

```
┌──────────────────────────┐
│ ← 3px 彩色条（蓝/橙/绿/灰）  │
│  ┌──┐                     │
│  │● │ 标题文字              │
│  └──┘                     │
│                           │
│  卡片内容区域               │
│                           │
└──────────────────────────┘
```

**实现方式**：使用 `border-top: 3px solid [color]` 或 `border` 属性。

### 3.2 标题彩色圆点

每个卡片的标题前放置 6px 的彩色圆点，颜色与顶部色条一致。

**实现方式**：圆点用 `Circle()` 或小 `Stack()` 组件，宽高 6px，`borderRadius: 3`。

### 3.3 图标背景色分类

九宫格、功能列表、缩略图等图标区域，背景色按分类着色：
- 学术类：`#E8F4FF`（浅蓝），图标填充 `#1677FF`
- 生活类：`#FFF3E8`（浅橙），图标填充 `#FF7D00`
- 完成类：`#E8FFEC`（浅绿），图标填充 `#00B42A`

图标尺寸保持：九宫格 32px、列表图标 24px、缩略图 80px/64px 不变。

### 3.4 分隔线统一

分隔线颜色统一从 `#E5E6EB` 改为更柔和的 `#F0F0F5`（新增辅助色 `divider_light`）。

## 七、实现策略

### 7.1 分支策略

从 `main` 创建新分支 `feat/frontend-design-polish`，所有改动在该分支上提交。

### 7.2 实现顺序

1. **基础设施**：color.json 新增 4 个辅助色 + 准备全局样式常量
2. **首页（HomeTab）**：九宫格 + 公告卡片 + 天气卡片
3. **校园信息（CampusInfoTab）**：标签 + 列表项
4. **学习助手（StudyTab）**：标签 + 课程卡片 + 成绩概览
5. **生活服务（LifeServiceTab）**：入口卡片 + 失物列表
6. **我的（MineTab）**：头像 + 功能列表
7. **登录/注册**：输入框聚焦态
8. **全部子页面**：分隔线统一 + 卡片样式对齐
9. **Review & 自检**：逐页确认视觉一致性

### 7.3 需新增 color.json 条目

```json
{
  "name": "card_bg_blue",
  "value": "#E8F4FF"
},
{
  "name": "card_bg_orange",
  "value": "#FFF3E8"
},
{
  "name": "card_bg_green",
  "value": "#E8FFEC"
},
{
  "name": "divider_light",
  "value": "#F0F0F5"
}
```

### 7.4 深色模式适配原则

所有新增浅色背景色（`#E8F4FF`、`#FFF3E8`、`#E8FFEC`）在深色模式下统一回退为对应的深色卡片背景色（当前项目深色卡背景为 `#252540` 或类似深色）。分隔线 `#F0F0F5` 在深色模式下回退为 `#3A3A5C`。
